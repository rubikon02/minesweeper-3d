# Minesweeper 3D

Multiplayer Minesweeper 3D game created in 10 days

[Play online](https://rubikon02-minesweeper.herokuapp.com/)


# Technologies

Server:
- Kotlin
- Spark
- PostgreSQL
- Heroku
- WebSocket

Client:
- JavaScript
- Three.js
